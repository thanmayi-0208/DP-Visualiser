# DP-Visualiser
DP Visualizer

A simple and interactive Dynamic Programming Visualizer built using HTML, CSS, and JavaScript.
This project helps users understand how common DP algorithms work step-by-step through visualization.

Features
Visualize multiple DP algorithms:
Fibonacci
Knapsack
Longest Common Subsequence (LCS)
Interactive input fields for custom test cases
Step-by-step grid visualization
Beginner-friendly UI
 Tech Stack
HTML
CSS
JavaScript
 Project Structure
DP-Visualizer/
│── index.html
│── style.css
│── script.js
│── sleep.js
│── fibonacci.js
│── knapsack.js
│── lcs.js
 How It Works
Select an algorithm from the dropdown
Enter required inputs:
Fibonacci → Enter n
Knapsack → Enter weights, values, and capacity
LCS → Enter two strings
Click Visualize
The grid shows how DP table is built step-by-step
 How to Run
Download or clone the repository
 Example Inputs
Fibonacci:
n = 6
Knapsack:
Weights: 1 2 3
Values: 10 15 40
Capacity: 6
LCS:
String1: ABCBDAB
String2: BDCAB
